# 更新日志

## [1.1.0] - 2026-06-24

### 新增
- 任务添加可选的"描述"栏位，小字显示在标题下方
- 点击任意任务可弹出编辑框修改标题和描述
- FAB 加号按钮（右下角）替代顶部输入框，点击弹窗添加任务（标题+描述）
- 信箱图标（右上角）替代文字按钮进入历史页
- 黑白风格图标（ic_add / ic_mail）

### 变更
- 主题改为 NoActionBar，去掉顶部黑色 ActionBar
- 整体色调改为浅蓝色背景 (#E1F5FE)
- 衍生按钮不再要求任务已完成，任何无子任务的任务都可直接衍生
- 衍生任务弹窗支持同时输入标题和描述

### 技术
- TodoItem 模型增加 `description` 字段
- 新增 dialog_add_todo.xml 和 dialog_edit_todo.xml 布局
- 适配器新增 `onEditItem` 回调
- 版本号从 1.0 → 1.1.0

## [1.0.0] - 初始版本

- 基础 Todo 任务的添加、完成、删除
- 任务衍生（子任务）功能
- 任务折叠/展开
- 历史记录查看与日期筛选
